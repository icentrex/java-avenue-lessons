## mentor1


Программа для учёта выданной техники пользователям

Примерный план работы программы

```cmd
Начальное меню:
1 - Пользователи
2 - Техника

При выборе 1:
  Вывести список созданных пользователей
  Доступные команды
  1 - Добавить пользователя
  2 - Выбрать пользователя
  
    При выборе 1:
    Заполнить необходимые поля, создать пользователя. Выбрать пользователя (как при варианте 2)
  
    При выборе 2:
    Информация о пользователе + закреплённая за ним технгика
    Команды
    1 - Изменить пользователя
    2 - Закрепить технику (Вывести список доступной техники с id)
    3 - Открепить технику
    4 - Удалить пользователя (Ошибка если есть закреплённая техника)
  
При выборе 2:
  Вывести список доступной техники
  Доступные команды
  1 - Добавить технику
  2 - Выбрать технику
  
    При выборе 1:
    Заполнить необходимые поля, создать технику. Выбрать технику (как при варианте 2)
    
    При выборе 2:
    Информация о технике + пользователь за кем техника закреплена
    Команды
    1 - Изменить технику
    2 - Закрепить технику (Вывести список пользователей с id)
    3 - Открепить технику
    4 - Удалить технику (Ошибка если техника закреплена)

```
